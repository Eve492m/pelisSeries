# 📖 Manual de Usuario — TKromas

Guía completa para usar la aplicación de consola PelisSeries.

## Primer uso

Al ejecutar el programa verás la pantalla de bienvenida seguida del menú de acceso:

- **Opción 1** — Iniciar sesión (si ya tienes cuenta)
- **Opción 2** — Crear nueva cuenta
- **Opción 0** — Salir

## Crear una cuenta

Elige un nombre de usuario único y una contraseña. El sistema pedirá confirmar la contraseña para evitar errores. La contraseña se guarda cifrada.

## Agregar una película o serie

Selecciona la opción 1 del menú principal e ingresa:

- **Título** — nombre del contenido
- **Tipo** — 1 para película, 2 para serie
- **Género** — Acción, Drama, Terror, etc.
- **Año** — entre 1888 y el año actual
- **Calificación** — del 1 al 5 (estrellas)

## Buscar contenido

Usa la opción de búsqueda para encontrar por:

- Título (búsqueda parcial)
- Género
- Año de estreno
- Calificación mínima

## Ver estadísticas

El sistema muestra:

- Total de películas y series registradas
- Género más frecuente en tu catálogo
- Promedio de calificaciones
- Contenido mejor calificado

## Exportar lista

## Cerrar sesión

Selecciona la opción **0** en el menú principal para salir de forma segura.
